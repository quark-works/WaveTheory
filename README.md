# WaveTheroy

Wireless column staggered PCB compatible with GH60 tray mount cases.

<p align="center">
  <img src="https://github.com/quark-works/WaveTheory/blob/main/Images/build.png?raw=true" />
</p>

## Features
- 12x4 column staggered board with space for two optional encoders.
- Powered by a e73-2G4M08S1C module, nrf52840 and running ZMK. Firmware available here: [wavetheory-zmk-config](https://github.com/quark-works/wavetheroy-zmk-config).
- Board is in Full HV mode, with a battery of 110 mAh the expected life is around 1 month.
- Has space for a nice!view to show information regarding layers, battery, etc.
- Compatible with GH60 tray mount cases (as pictured).
